# 📊 Sales Dashboard

This project is an interactive Sales Dashboard built using Python, Streamlit, Pandas, and Plotly.

## 🚀 Features

* Interactive filters (Region & Date)
* Sales analysis with charts
* Clean and user-friendly interface
* Real-time data visualization

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* Plotly

## 📂 Files

* app.py → Main dashboard code
* sales_data.csv → Dataset
* requirements.txt → Dependencies

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 🌐 Deployment

You can deploy this project using Streamlit Cloud.
